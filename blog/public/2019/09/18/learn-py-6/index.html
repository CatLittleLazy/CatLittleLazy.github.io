使用模块

  • 将函数实现在不同的文件中，使用时导入对应的模块即可
  • 作用域可以通过模块中方法名前添加_实现私有成员变量或方法，但是由于python中没有实现private所以其实还是可以访问，只是以此作为标志而已

安装第三方模块(windows下)

  • python官网下载的zip解压版会有问题，需要重新下载Windows x86 executable installer版本
  • 下载后重新配置环境变量，配置完成后重启生效环境变量
  • 打开cmd进入python37/Scripts(也可以讲pip加入环境变量中)
  • 输入pip install pillow安装第三方模块

常用第三方模块

  • Pillow

  • requests

  • chardet(解决编码问题)(没觉得有什么用)

  • psutil(process and system utilities主要用于获取系统信息)
